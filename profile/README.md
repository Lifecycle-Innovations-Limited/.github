# Lifecycle Innovations Limited

Lifecycle Innovations Limited builds AI-powered health and productivity tools.

## Products

### Healify

AI health companion mobile app available on iOS and Android.

Built with React Native/Expo, NestJS, and LangGraph, Healify delivers health tracking, AI-powered insights, personalized recommendations, and HealthKit integration.

- [App Store](https://apps.apple.com/app/healify)
- [Website](https://www.healify.ai)

### claude-ops

Open-source Claude Code plugin that turns Claude into a business operating system.

Features: morning briefings, unified inbox, autonomous PR merge, and infrastructure monitoring.

- [View on GitHub](https://github.com/Lifecycle-Innovations-Limited/claude-ops)
